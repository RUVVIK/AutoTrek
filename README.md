# AutoTrek

**Система планирования и контроля расходов автопарка**

WPF-приложение для автоматизации учёта и анализа затрат на содержание транспортных средств.

## 📋 Описание

AutoTrek — это полнофункциональная информационная система для управления автопарком, разработанная в рамках курсовой работы. Приложение предоставляет разграниченный доступ для различных ролей сотрудников и позволяет вести учёт путевых листов, заправок, ремонтов, а также формировать детальные отчёты и аналитику.

## ✨ Возможности

### 📊 Учёт и регистрация
- Ввод данных с путевых листов
- Регистрация заправок транспортных средств
- Оформление ремонтов и технического обслуживания
- Ведение истории обслуживания ТС

### 📈 Аналитика и отчётность
- Отчёты по затратам автопарка
- Детальный анализ расходов по ТС
- Сравнительный анализ план/факт
- Визуализация отклонений и перерасходов
- Динамика затрат по месяцам

### 💰 Управление ценами
- Редактирование цен на топливо
- Управление стоимостью запчастей и услуг
- Отслеживание средних цен за период

## 👥 Роли пользователей

### Диспетчер
- Ввод путевых листов
- Регистрация заправок
- Просмотр календаря ТО

### Механик
- Оформление ремонтов и ТО
- Заявки на приобретение запчастей
- Просмотр истории обслуживания ТС

### Бухгалтер
- Формирование ведомостей по затратам
- Редактирование цен и тарифов
- Детальная аналитика по ТС

### Руководитель автопарка
- Дашборд с ключевыми показателями (KPI)
- ТОП-5 самых затратных ТС
- Предстоящие ТО
- Утверждение бюджета
